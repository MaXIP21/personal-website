Automated AWS deployment for your personal website

Posted by Peter Bacsai Nov 29, 2021 AWS

Creation of an automated deployment for your CV website can feel a bit overwhelming, especielly when you hear from this method at the first time. By using AWS tools it's more easy than you think, don't be scared we will describe all the necessary steps to create a complete end to end deployment to Amazon EC2. Please also note this post is not describing the setup process of your git repository, as this is a very commonly used software and there is a bunch of great tutorials how you can use and benefit from GitHub.

The tools will use in this tutorial:

  • Visual Studio Code
  • GitHub
  • AWS CodePipeline and Codedeploy
  • Amazon EC2 instance to host your website
  • Namecheap for the Domain registration

So let's begin with Visual Studio Code (VSCode).

Visual Studio Code is a great product of Microsoft, this is by far one of the most popular IDE for Python, Java, or even web development. The great news is that it can run on Windows, Linux and Mac as well so you don't need to install another operating system if you use only Linux for example. You can freely download Visual Studio Code from microsoft's official site here: https://visualstudio.microsoft.com/downloads/

After the installation completed run the software and create a new workspace or use the File menu "Add Folder to Workspace" option to import your existing html page. First thing to do is to set up source control "GitHub" in order to push our website surce code there. With the help of GitHub we can follow up the lifecycle of a software and roll back in case something went wrong. There is an official tutorial how you can configure VSCode to sync to your git repository. https://code.visualstudio.com/docs/editor/github
