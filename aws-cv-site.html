Automated AWS deployment for your personal website

Posted by Peter Bacsai Nov 29, 2021 AWS

Creation of an automated deployment for your CV website can feel a bit overwhelming, especielly when you hear from this method at the first time. By using AWS tools it's more easy than you think, don't be scared we will describe all the necessary steps to create a complete end to end deployment to Amazon EC2.

The tools will use in this tutorial:

  • Visual Studio Code
  • GitHub
  • AWS CodePipeline and Codedeploy
  • Amazon EC2 instance to host your website
  • Namecheap for the Domain registration
